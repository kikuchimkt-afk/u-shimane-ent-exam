# 島根県立大学 国際関係学部 入試対策ポータル

## 概要
島根県立大学 国際関係学部の小論文試験対策用Webポータルです。
過去問や模擬試験のPDFダウンロード、および対策ガイドを提供します。

## 機能
- **レスポンシブデザイン**: スマホ・PCに対応したモダンなUI
- **試験アーカイブ**: 過去問および5つの模擬試験ページ
- **PDFダウンロード**: 各試験の問題冊子・指導マニュアル・解答例へのリンク
- **対策ガイド**: 合格のための戦略とテーマ解説

## デプロイ方法 (Vercel)
1. GitHubリポジトリにこのフォルダの内容をプッシュします。
2. Vercelのダッシュボードで「Add New Project」を選択します。
3. GitHubリポジトリをインポートします。
4. Framework Presetは「Other」（またはHTML/CSS）で自動認識されます。
5. 「Deploy」をクリックして完了です。

## 注意事項
- PDFファイル名に日本語が含まれています。Vercelでは通常問題ありませんが、文字化け等の不具合が出る場合は英数字（例: `exam_r7.pdf`）への変更を検討してください。
